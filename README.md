# MadLadSquadBot
The official discord bot for the MadLadSquad discord server
## Features
- Serverinfo
- Userinfo
- Help
- Playing music in vc
- Ban
- Kick
- Event logs
- Welcome images
- Automatic moderation
- Reaction roles
- Restricted channels
- Topic locked channels
- Twitch, Twitter and Youtube updates
- Activity leaderboard
